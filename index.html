Animated TY text renderd as GIF in the browser, using canvg and gifenc, artistic inspiration from discordmojis.com/emojis/2905-thankyou. This one creates a “party” or “disco” effect, meaning that the background color (hue) changes over time. The text, font size, and animation speed can be configured. All processing is done in the browser, no data is collected.
